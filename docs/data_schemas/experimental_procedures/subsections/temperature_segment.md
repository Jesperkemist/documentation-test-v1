??? "time_of_step"
    Time of the step.

    * type: float
    * unit: minute
    * example: 10

??? "temperature_start"
    Temperature at the start of the step.

    * type: float
    * unit: C
    * example: 50    

??? "temperature_end"
    Temperature at the end of the step.

    * type: float
    * unit: C
    * example: 200   

??? "temperature_acceleration"
    Temperature acceleration.

    * type: float
    * unit: C/min
    * example: 200
