??? "time_of_segment"
    Time of the segment.

    * type: float
    * unit: minute
    * example: 10

??? "temperature_start"
    Temperature at the start of the segment.

    * type: float
    * unit: C
    * example: 50    

??? "temperature_end"
    Temperature at the end of the segment.

    * type: float
    * unit: C
    * example: 200   

??? "temperature_acceleration"
    Temperature acceleration.

    * type: float
    * unit: C/min
    * example: 200
