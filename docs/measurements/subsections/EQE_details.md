??? "wavelength_at_start"
    Wavelength at start of measurement 

    * type: float
    * unit: nm
    * example: 320 

??? "wavelength_at_end"
    Wavelength at end of start of measurement 

    * type: float
    * unit: nm
    * example: 950    

??? "step_size"
    The size of the steps in the scanning

    * type: float
    * unit: nm
    * example: 5  

??? "integration_time"
    Integration time for each wavelength

    * type: float
    * unit: s
    * example: 2      

??? "scan_speed"
    Scan speed.

    * type: float
    * unit: nm/s
    * example: 2.5 

??? "equipment"
    Brand name and model of the equipment used for the measurement.

    * type: str
    * example: "Newport"
